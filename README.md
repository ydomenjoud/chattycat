# Chattycat website
![BUILD](https://github.com/ydomenjoud/chattycat/workflows/BUILD/badge.svg)

## website
`cd website`

* install: `npm i`
* serve: `npm run website:serve`
* build: `npm run website:build`

## functions
`cd functions`

* serve: `npm run serve`
* logs: `npm run logs`
* deploy: `npm run deploy`
